Welcome to Super Seven's Project Management System (SSPMS):

Our easy to use website provides software development teams with the tools they need to quickly and accurately delegate tasks. Our task system is based off a kanban-like interface where users can create, update, and edit tasks within three main categories - ToDo, Doing, Done. Tasks in any category contains the information of who has taken responsibility for completeing the task along with an estimated time to completion. Each task can seamlessly be moved from one category to another as needed by the user.
